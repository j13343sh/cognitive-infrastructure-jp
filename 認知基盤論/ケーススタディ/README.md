# ケーススタディ

認知基盤論は、理論だけで構成されているRepositoryではない。

本Repositoryでは、実際の研究活動や組織運営を通じて得られたObservationも重要な情報源として位置付けている。

本ディレクトリに収録されているケーススタディは、認知基盤論の各理論を証明することを目的としたものではない。

限定された研究環境において実際に観測されたRealityを整理し、どのような試行錯誤を経て現在の理論へ至ったのかを記録する実践資料である。

そのため、ここで示される内容は普遍的な結論ではなく、Current Working DraftにおけるObservationとして保存されている。

## Repository Philosophy

本ケーススタディは、理論を正当化するために存在するものではない。

理論とRealityが矛盾した場合、本Repositoryは理論ではなくRealityを優先する。

そのため、本ディレクトリには成功だけでなく、失敗や試行錯誤もObservationとして保存される。

Future Generationが過去のDecision Contextを再構築し、再びRealityとの対話を開始できることを、本Repositoryは重視している。
